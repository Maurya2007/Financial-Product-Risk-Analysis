# Financial Product Risk Analysis

## Objective
Analyze customer and product usage data to identify engagement patterns, potential risk signals, and opportunities for product improvement.

## Dataset
Synthetic financial product dataset representing customer demographics, usage behavior, and engagement metrics.

## Method
- Data cleaning and exploratory analysis using Python
- Customer segmentation using clustering
- Identification of risk indicators based on engagement and usage patterns
- Visualization of insights to support product and risk decisions

## Key Outputs
- Customer segments
- Engagement and risk indicators
- Visual summaries and recommendations

## Findings
- Identified distinct customer segments with varying engagement levels
- Highlighted behaviors correlated with higher churn and risk
- Surfaced opportunities to improve retention through targeted interventions

## Recommendations
- Focus on early engagement signals to reduce churn
- Personalize product experiences for high-risk segments
- Monitor key metrics continuously to support data-driven decisions

## How to Run
1. Install dependencies from `requirements.txt`
2. Run analysis scripts in the `src/` folder
